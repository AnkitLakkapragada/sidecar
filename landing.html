Sidecar watches your agent's trajectory, reflects on every drift, and writes lessons the agent reads on the next session — so the pressure that broke it once doesn't break it twice.

the loop
01 / supervise

Watch the trajectory.

Four signals after every turn: drift, sycophancy, persona collapse, memory poisoning. Tripped → automatic intervention.

02 / reflect

Turn drift into lessons.

After the session, a therapist model rewrites every drifted turn in-character. Pattern + correction + rationale = a Lesson.

03 / remember

Carry the memory.

Lessons live in SQLite + FTS. Next session opens with the relevant ones injected. The agent walks in pre-warned.

Same script. Same model. Therapy in between.
